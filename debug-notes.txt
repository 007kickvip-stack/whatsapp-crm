问题1 (已修复): byStaff 查询 ORDER BY totalProfit DESC - 别名不存在
问题2 (新发现): dailyTrend 查询 ORDER BY date ASC - 别名 date 不存在

两个问题根源相同：Drizzle ORM 的 select 中使用 sql`` 模板生成的列没有在SQL中生成别名，
所以 ORDER BY 引用的别名在实际SQL中不存在。

修复方案：
- byStaff: 已改为 ORDER BY COALESCE(SUM(...), 0) DESC
- dailyTrend: 需要改为 ORDER BY DATE_FORMAT(...) ASC

另外，还需要将整个函数改为分步执行，避免一个查询失败导致整个函数返回500。
或者更好的方案：将三个查询分别 try-catch，确保部分查询失败不影响其他数据。
